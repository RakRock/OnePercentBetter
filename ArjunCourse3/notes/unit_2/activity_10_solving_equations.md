## Quick Review Notes

### Main Idea
An **equation** says two expressions are **equal**. To solve, use **inverse operations** so the variable is alone. Think of a **balance scale**: whatever you do to one side, do to the other to stay balanced.

### Key Vocabulary
- **Equation:** Two expressions joined by `=`.
- **Solution:** The value of the variable that makes the equation true.
- **Inverse operation:** Undoes an operation (add ‚Üî subtract, multiply ‚Üî divide).
- **Like terms:** Terms with the same variable part (e.g. `3x` and `5x`).
- **Distributive Property:** `a(b + c) = ab + ac`.
- **No solution:** Equation is never true (e.g. `2 = 5`).
- **Infinitely many solutions:** Every value works (same line on both sides).

### Important Formulas / Rules

**Remember ‚Äî balance idea:**

| To undo‚Ä¶ | Do this to both sides |
|----------|------------------------|
| `+ a` | subtract `a` |
| `‚àí a` | add `a` |
| `√ó a` (a ‚âÝ 0) | divide by `a` |
| `√∑ a` (a ‚âÝ 0) | multiply by `a` |

**Order of solving (typical):**
1. Distribute if needed  
2. Combine like terms on each side  
3. Move variable terms to one side  
4. Move numbers to the other side  
5. Divide or multiply to isolate the variable

### Example 1

**Problem:** Solve `3x + 4 = 16`.

[DIAGRAM:solve_steps]

**Solution:**
1. Subtract 4: `3x = 12`  
2. Divide by 3: `x = 4`  
3. Check: `3(4) + 4 = 16` ‚úì  

**Answer:** **`x = 4`**

### Example 2

**Problem:** Solve `5x + 2 = 2x + 11`.

**Solution:**
1. Subtract `2x` from both sides: `3x + 2 = 11`  
2. Subtract 2: `3x = 9`  
3. Divide by 3: `x = 3`  
4. Check in original equation ‚úì  

**Answer:** **`x = 3`**

### Special cases: no solution vs infinitely many

Sometimes solving does **not** give one value for `x`. After you simplify, look at what is left:

| What you get | Meaning |
|--------------|---------|
| `x =` a number | **One solution** (like Examples 1 and 2) |
| A **false** statement (e.g. `1 = 5`) | **No solution** ‚Äî no value of `x` works |
| A **true** statement with no variable (e.g. `6 = 6`) | **Infinitely many solutions** ‚Äî every `x` works |

### Example 3 ‚Äî No solution

**Problem:** Solve `2x + 1 = 2x + 5`.

**Solution:**
1. Subtract `2x` from both sides: `1 = 5`  
2. Stop ‚Äî `1 = 5` is **never true**, no matter what `x` is.  
3. The `x` terms canceled, but the numbers do **not** match.

**Answer:** **No solution** (sometimes written as **‚àÖ**, ‚Äúempty set‚Äù).

**Why it happens:** Both sides have the same `x`-part (`2x`) but different constants (`1` and `5`). It is like saying ‚Äúthe same unknown amount plus 1 equals the same unknown amount plus 5,‚Äù which is impossible.

#### More examples ‚Äî no solution

**A.** `3x ‚àí 4 = 3x + 2`  
Subtract `3x`: `‚àí4 = 2` ‚úó ‚Üí **no solution**

**B.** `5x + 7 = 5x ‚àí 1`  
Subtract `5x`: `7 = ‚àí1` ‚úó ‚Üí **no solution**

**C.** `4(x + 2) = 4x + 10`  
Distribute: `4x + 8 = 4x + 10` ‚Üí subtract `4x`: `8 = 10` ‚úó ‚Üí **no solution**

**D.** `‚àí2x + 9 = ‚àí2x + 3`  
Add `2x` to both sides: `9 = 3` ‚úó ‚Üí **no solution**

**E.** `6x + 1 = 2x + 1 + 4x`  
Combine right side: `6x + 1 = 6x + 1` ‚Üí subtract `6x`: `1 = 1` ‚úì  

Wait ‚Äî that last one is **infinitely many**, not no solution! Always simplify **fully** before you decide.

### Example 4 ‚Äî Infinitely many solutions

**Problem:** Solve `2(x + 3) = 2x + 6`.

**Solution:**
1. Distribute on the left: `2x + 6 = 2x + 6`  
2. Subtract `2x` from both sides: `6 = 6`  
3. Stop ‚Äî `6 = 6` is **always true** for **every** value of `x`.  
4. You cannot isolate a single `x`; any number you try will work.

**Answer:** **Infinitely many solutions** (all real numbers).

**Check with two different values:**
- If `x = 0`: left `2(0 + 3) = 6`, right `2(0) + 6 = 6` ‚úì  
- If `x = 4`: left `2(4 + 3) = 14`, right `2(4) + 6 = 14` ‚úì  

**Why it happens:** After simplifying, both sides are the **same expression**. The equation does not pin down one `x`; it is true for all `x`.

#### More examples ‚Äî infinitely many solutions

**A.** `4x + 2 = 4x + 2`  
Subtract `4x`: `2 = 2` ‚úì ‚Üí **infinitely many solutions**

**B.** `3(x ‚àí 1) = 3x ‚àí 3`  
Distribute: `3x ‚àí 3 = 3x ‚àí 3` ‚úì ‚Üí **infinitely many solutions**

**C.** `5x ‚àí 10 = 5(x ‚àí 2)`  
Right side: `5x ‚àí 10 = 5x ‚àí 10` ‚úì ‚Üí **infinitely many solutions**

**D.** `‚àíx + 4 = 4 ‚àí x`  
Both sides are the same (order does not matter) ‚Üí **infinitely many solutions**

**E.** `2x + 8 = 2(x + 4)`  
Distribute: `2x + 8 = 2x + 8` ‚úì ‚Üí **infinitely many solutions**

**F.** `6x + 1 = 2x + 1 + 4x`  
Combine like terms on the right: `6x + 1 = 6x + 1` ‚úì ‚Üí **infinitely many solutions**

**Quick check:** Pick any `x` (like `0`, `1`, or `‚àí2`) and plug into the **original** equation. If it works for one value and the variables canceled to a true statement, it works for **all** values.

### Common Mistakes
- Changing only **one side** of the equation.
- Sign errors when moving terms (subtracting a negative).
- Dividing by the **variable** instead of the coefficient.
- Forgetting to **check** the answer in the original equation.
- Thinking `x = 0` is ‚Äúno solution.‚Äù
- Stopping at `6 = 6` and saying ‚Äú`x = 6`‚Äù ‚Äî **`6 = 6` means infinitely many**, not `x = 6`.
- Giving up when variables cancel ‚Äî check whether you get a **false** or **true** number statement.

### Mini Summary
- Keep the equation **balanced** on both sides.
- Use **inverse operations** to isolate the variable.
- **Distribute** and **combine like terms** first when needed.
- **No solution** ‚Üí false statement; **infinitely many** ‚Üí always true.
- Always **substitute back** to check.
