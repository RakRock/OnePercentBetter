## Unit formulas cheat sheet

| Idea | Formula |
|------|---------|
| Constant difference pattern | `nth = first + diff(n − 1)` |
| Square numbers | `n²` |
| Triangular numbers | `n(n+1)/2` |
| Slope | `(y₂−y₁)/(x₂−x₁)` |
| Slope-intercept | `y = mx + b` |
| Proportional | `y = kx` |
| System solution | `(x, y)` in **both** equations |

## Before a quiz — quick checklist

- [ ] Can you write an expression from a table or dot pattern?
- [ ] Can you solve `ax + b = c` and equations with variables on both sides?
- [ ] Can you find slope from two points or a table?
- [ ] Can you identify `m` and `b` in `y = mx + b` and graph it?
- [ ] Can you tell proportional vs. nonproportional?
- [ ] Can you find a system solution by graphing or algebra?
- [ ] Do you check answers by substituting back?
