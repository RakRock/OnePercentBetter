[KEY]

## Quick Review Notes

### Main Idea
A **system** is two (or more) linear equations using the **same variables**. A **solution** is an ordered pair `(x, y)` that makes **both** equations true — usually the **point where the lines cross** on a graph.

### Key Vocabulary
- **System of equations:** Two or more equations together.
- **Solution (of a system):** The `(x, y)` that satisfies every equation.
- **Intersection:** Where two graphs meet.
- **Parallel lines:** Same slope, different y-intercepts → **no solution**.
- **Coinciding lines:** Same line twice → **infinitely many solutions**.

### Important Formulas / Rules

**Remember:**

- **One solution:** Lines intersect at one point.
- **No solution:** Lines are **parallel** (same slope, different intercepts).
- **Infinitely many:** Lines are the **same** (equivalent equations).

[DIAGRAM:three_cases]

**Solving by graphing:**
1. Graph each equation (often slope-intercept form).  
2. Find the **intersection point**.  
3. Read **(x, y)** — check in both equations.

### Visual Explanation

The **solution** is the intersection point — it must work in **both** equations.

### Example 1

**Problem:** Solve by graphing: `y = x + 1` and `y = −x + 5`.

[DIAGRAM:intersection]

**Solution:**
1. Line 1: slope 1, y-intercept 1  
2. Line 2: slope −1, y-intercept 5  
3. Intersection appears at **(2, 3)**  
4. Check: `3 = 2 + 1` ✓ and `3 = −2 + 5` ✓  

**Answer:** **(2, 3)**

### Example 2

**Problem:** How many solutions does the system `y = 3x + 2` and `y = 3x − 1` have?

[DIAGRAM:parallel]

**Solution:**
- Both have slope **3** but different intercepts (2 and −1)  
- Lines are **parallel**  

**Answer:** **No solution**

**Problem:** Itmar: 63 in. in August, grows **½ in./month**. Megan: 65 in., grows **¼ in./month**. Equations? Same height?

**Solution:**
1. `I = 63 + 0.5m`, `M = 65 + 0.25m`

### Common Mistakes
- Picking a point on **only one** line (must work in **both**).
- Graphing inaccurately so the intersection looks wrong — **check algebraically** when possible.
- Thinking **parallel** lines have one solution.
- Forgetting to write the answer as an **ordered pair** `(x, y)`.

### Mini Summary
- Solution = **intersection** of the graphs.
- **One** intersection → one solution; **parallel** → none; **same line** → infinitely many.
- Graph both lines carefully or use a **table** of values.
- Always **check** `(x, y)` in both equations.
