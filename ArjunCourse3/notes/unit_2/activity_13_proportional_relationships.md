## Quick Review Notes

### Main Idea
A **proportional relationship** is a special linear pattern where `y` is always a **constant multiple** of `x`. The graph passes through the **origin (0, 0)** and the equation looks like **`y = kx`** (no extra number added).

### Key Vocabulary
- **Proportional:** `y` and `x` grow together from zero; ratio `y/x` is constant.
- **Nonproportional:** Has a starting amount when `x = 0` (y-intercept ≠ 0).
- **Constant of proportionality (k):** The number you multiply `x` by to get `y`.
- **Direct variation:** Another name for proportional (`y = kx`).
- **Unit rate:** Often the value of `k` (e.g. dollars per pound).

### Important Formulas / Rules

**Remember:**

`y = kx`   (proportional — no `+ b`)

| Check | Proportional? |
|-------|----------------|
| Graph through **(0, 0)** | Yes |
| Equation has **only** `kx` (b = 0) | Yes |
| Table: `y/x` same every row (x ≠ 0) | Yes |
| Equation `y = 2x + 5` | **No** (b ≠ 0) |

`k = y / x`  (when x ≠ 0)

### Visual Explanation

**Proportional** graphs pass through the origin. **Nonproportional** lines cross the y-axis above or below zero.

### Example 1

**Problem:** Does this table show a proportional relationship?

| x | 2 | 4 | 6 |
|---|---|---|---|
| y | 7 | 14 | 21 |

[DIAGRAM:table_constant]

**Solution:**
- Ratios: `7/2 = 3.5`, `14/4 = 3.5`, `21/6 = 3.5` — all equal  
- If `x = 0`, `y` would be 0  
- Equation: **`y = 3.5x`**  

**Answer:** **Yes**, proportional with **k = 3.5**.

### Example 2

**Problem:** Is `y = 4x + 1` proportional?

**Solution:**
- When `x = 0`, `y = 1` (not 0)  
- There is a **+ 1** (nonzero y-intercept)  

**Answer:** **No** — it is linear but **not proportional**.

**Problem:** 330 miles in 5.5 hours. How many hours for 720 miles? (Assume proportional distance vs. time.)

### Common Mistakes
- Thinking **any straight line** is proportional (need through origin).
- Using **`y = mx + b`** when `b` must be **0** for proportional.
- Dividing **`x/y`** instead of **`y/x`** to find `k`.
- Confusing **proportional** with **linear**.

### Mini Summary
- Proportional → **`y = kx`**, graph through **(0, 0)**.
- **k** is the constant of proportionality / unit rate.
- **+ b** with b ≠ 0 → **not** proportional.
- Check tables with **y/x** (same ratio) or graph for the origin.
